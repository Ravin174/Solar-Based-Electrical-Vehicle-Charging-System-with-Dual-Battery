         explain the project clearly:  Solar Based Electrical Vehicle Charging System with Dual BatteryOverviewThis project, published in JETIR (Volume 12, Issue 2, Feb 2025) ,
             addresses "range anxiety" in Electric Vehicles (EVs) by combining Wireless Power Transfer (WPT) with an intelligent Dual Battery Management System.

    Key FeaturesDynamic Wireless Charging:     Allows power transfer via coils while the vehicle is in motion.
    Dual Battery Setup:   Utilizes a primary and a backup battery to ensure an uninterrupted power supply.
    Automated Switchover:   A microcontroller-based system that toggles batteries based on real-time voltage levels (Thresholds: 3.6V and 3.0V).
    Solar Integration:   Uses solar panels as the primary green energy source to charge the batteries.
    Hardware StackMicrocontroller:   ATmega328P (Arduino Uno).Sensors: Voltage sensors for real-time monitoring.
    Actuators:   Relay modules for physical battery switching.Display: 16x2 LCD for system status and battery levels.
    Power:     Solar Panel, Copper Coils (for WPT), and 12V Rechargeable Batteries.
